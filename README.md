### 실행 방법
1. .env setting
```
OPENAI_API_KEY=OPEN_API_TOKEN
MAPBOX_API_TOKEN=MAPBOX_API_TOKEN
```

2. Show Dashboard by streamlit run
- `chmod +x run.sh`
- run code by entering `./run.sh` or `python -m streamlit run 홈페이지.py`
